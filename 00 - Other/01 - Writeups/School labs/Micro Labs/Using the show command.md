# Description
In this lab, you will learn to use the `show` command. The `show` command lists the currently known facts about the switchâ€™s operational status. The only work the switch does in reaction to `show` commands is to find the current status and list the information in messages sent to the user.

# Process

2. Enter the given IP address and port, select Telnet, and press open
3. Enter `en` to go to enable mode
